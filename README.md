## 使用方法
默认模板文件由以下四部分组成：

- `main.tex` 主文件,包含报告正文
- `reference.bib` 参考文献列表
- `CQMUReport.sty` 文档格式控制
- `figures` 放置图片的文件夹，模板中现在包含院校LOGO和效果预览图。

使用时只需要修改 `main.tex` 和 `reference.bib` 即可，论文插图可以放在 `figures` 文件夹下，模板可以自动识别常见的插图文件夹，如 `figures`、`image`、`img` 等。
